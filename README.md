# flutter_shop

JSPang flutter移动电商实战项目

## 目录结构

### lib/
* main.dart:入口文件
* config: 配置数据
* model: 数据model
* page: 导航目录页面
* provide: 状态管理
* service: dio网络封装相关
* ui: 公共widget，独立widget



### 依赖的第三方框架
 
  * cupertino_icons: ^0.1.2     # ios风格图标
  
  * dio: ^2.0.20    # 网络框架
  
  * cached_network_image: ^0.5.1    # 图片框架
  
  * flutter_swiper: ^1.1.4      # 无限轮播
  
  * flutter_screenutil: ^0.5.1      # 屏幕适配，写的一般
  
  * url_launcher: ^5.0.1     #
  
  * flutter_easyrefresh: ^1.2.7     # 下拉刷新，上拉加载更多
  
  * provide: ^1.0.2     # 状态管理
